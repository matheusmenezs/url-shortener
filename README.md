## Description
URL shortener created in Nest.js

## Requirements
- Typescript
- Node.js v22.11.0
- Docker
- Npm or Yarn
- docker-compose

## Environment variables
For the project to run properly, it is necessary to configure an environment variables file. In the root of the project there is a file called .env.example that can serve as a basis for your configuration.
Create a file called ".env" in the root of your project and define the variables as in the .env.example.

## Database 
```bash
$ docker-compose up -d
```

## Running Migrations
To update your database, run the command below. It will run the migrations related to the current schema. If there are changes in the schema.prisma, it will generate a new migration

```bash
npx prisma migrate dev
```
If you want to create a new migration, run the command below

```bash
npx prisma migrate dev --name migration_name
```

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
